# ssinit

# Server
```
Vultr, ubuntu 14.01
```

# Download
```
mkdir ssinit && cd ssinit && wget https://raw.githubusercontent.com/MKFons/ssinit/master/install.url && chmod 755 *.sh
```

# Install
```
./ss_init.sh
```

# Finish
```
./ss_done.sh
```
